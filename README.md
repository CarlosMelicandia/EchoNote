# EchoNote

EchoNote is a voice-powered task manager that lets you speak your tasks instead of typing them. It uses Googleâ€™s Speech-to-Text to transcribe your voice and Google Gemini AI to break it into clear, structured tasks. The app is built with Flask and stores tasks in a local SQLite database.

## Features

- Record voice to create tasks
- AI extracts and structures tasks automatically
- Tasks are stored and displayed in a clean web interface

## Tech Stack

- Python (Flask)
- HTML, CSS
- SQLite with SQLAlchemy
- Google Cloud Speech-to-Text
- Google Generative AI (Gemini)
